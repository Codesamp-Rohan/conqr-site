## Installation

1.  **Clone the repository to your local machine:**
    ```bash
    git clone https://github.com/codesamp-rohan/conqr-site.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd conqr-site
    ```

3.  **Install the project dependencies:**
    ```bash
    # If using npm
    npm install

    # If using Yarn
    yarn install
    ```

## Running the Site

**To run the development server:**

```bash
# Using npm
npm run dev

# Using Yarn
yarn dev
```

This command will typically start a local development server, often accessible at `http://localhost:[PORT]` (check your project's configuration for the exact port).
